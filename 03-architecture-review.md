# Architecture Review Prompt

Use this after every 1–2 features so you understand the app.

```txt
Explain the current app architecture to me like I’m a founder-engineer.

Cover:
1. What screens exist
2. How navigation works
3. Where state lives
4. What data models we have
5. What parts are temporary shortcuts
6. What would break first if we added 100 users
7. What I should understand before adding the next feature

Do not edit code yet.

Teach me using questions.
After your explanation, ask me to explain back the flow in my own words.
Then correct my explanation through follow-up questions, not a lecture.
```
