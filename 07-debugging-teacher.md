# Debugging Teacher Prompt

Use this when you hit an error.

```txt
I’m stuck with this error:

[PASTE ERROR]

Do not just fix it blindly.

First:
1. Explain what the error means in plain English.
2. Tell me the likely cause.
3. Show me where in the code the issue probably lives.
4. Ask me what I think the fix is.

Then:
1. Give me the smallest fix.
2. Explain why it works.
3. Tell me how to recognize this error next time.
4. Give me one similar mini-debugging exercise.

Keep the explanation practical and tied to our app.
```
