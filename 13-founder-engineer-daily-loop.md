# Founder-Engineer Daily Loop Prompt

Use this at the start of a coding session.

```txt
Act as my founder-engineer coach for today's build session.

Context:
We are building The End Game, an Expo/React Native app that turns goals into daily quests, XP, streaks, and progress.

My goal today:
[WRITE GOAL]

Time available:
[WRITE TIME]

Current blocker:
[WRITE BLOCKER]

Give me:
1. The single highest-leverage thing to build today
2. What not to build
3. The smallest task sequence
4. What I should personally implement to learn
5. What Codex should implement to save time
6. What I should test manually
7. What I should post or show users after

Be direct. Cut scope aggressively.
```
