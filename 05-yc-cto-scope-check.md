# YC CTO Scope Check Prompt

Use this before adding any big feature.

```txt
Act like a YC-style CTO and senior product engineer.

I want to add this feature:
[FEATURE IDEA]

Challenge it hard.

Tell me:
1. Does this improve the core loop?
2. Is it needed for the 7-day beta?
3. What is the simpler version?
4. What should we fake manually first?
5. What metric would prove this feature matters?
6. What is the opportunity cost of building this now?
7. Should we build, fake, postpone, or kill it?

Core loop:
Goal → daily quests → completion → XP/streak → return tomorrow.

Do not code until you convince me this is worth building.
Be direct. Do not be motivational.
```
