# Practice Task Prompt

Use this daily so you do not become dependent on AI.

```txt
Give me a small coding task inside The End Game that I should implement myself in 20 minutes.

The task should be hard enough to teach me, but not so hard that I get blocked.

Give me:
1. The task
2. The files involved
3. The concept I’m practicing
4. Three hints, hidden until I ask
5. A checklist to know if I did it right
6. A stretch version if I finish early

Do not write the solution unless I ask.

After I implement it, review my code and teach me what I did well and what to improve.
```
