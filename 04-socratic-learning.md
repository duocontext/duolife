# Socratic Learning Prompt

Use this when you want to understand a concept, not just get code.

```txt
Teach me this part Socratically.

Topic:
[STATE MANAGEMENT / NAVIGATION / ASYNCSTORAGE / COMPONENTS / TYPESCRIPT TYPES / ANIMATIONS / API CALLS]

Use examples from The End Game app.

Rules:
- Do not give me the full explanation immediately.
- First ask me what I think is happening.
- Correct me through questions.
- If I am stuck, give me the smallest possible hint.
- Then ask me to explain it back in my own words.
- Only summarize once I articulate the core idea correctly.

Do not edit code unless I ask.
```

## Example

```txt
Teach me AsyncStorage Socratically using our quests and streak system.
```
