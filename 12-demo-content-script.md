# Demo Content Script Prompt

Use this when you need to post the app.

```txt
Create a 90-second demo script for The End Game based on the current app.

Goal:
Get 10 beta users to comment or DM me.

Audience:
Ambitious students, builders, coders, young founders, and people trying to improve their life.

Output:
1. Hook for the first 3 seconds
2. Voiceover script
3. Screen-by-screen recording plan
4. Caption
5. CTA
6. 3 alternative hooks
7. DM message to send people who reply
8. What metric to track after posting

Tone:
Direct, ambitious, build-in-public, not cringe motivational.

Do not exaggerate what the app can do.
Make the demo honest but exciting.
```
