# UI Polish Prompt

Use this when the app works but needs to look demo-worthy.

```txt
Polish the UI of this screen for a 90-second product demo.

Screen:
[SCREEN NAME]

Goal:
Make it feel clean, modern, and emotionally aligned with The End Game: ambitious, game-like, but not childish.

Constraints:
- Do not change the core logic.
- Do not add unnecessary libraries.
- Keep the design shippable.
- Prioritize readability, spacing, hierarchy, and clear calls-to-action.
- Mobile-first.
- Avoid overcomplicated animations.

Before editing:
1. Explain what currently feels weak.
2. Explain the design direction.
3. List the files you will touch.

After editing:
1. Summarize the visual changes.
2. Explain the key React Native styling concepts.
3. Give me one small UI tweak to do myself.
4. Tell me how to test it on a real phone.
```
